BOOSE Programming Language

Basic Object Orientated Software Engineering

Project Description

This project implements a custom programming language called BOOSE. It supports variables, arrays, arithmetic operations, drawing commands, methods, loops, and conditionals.

Features

  • Variables (int, real, string)
  • Loops and If Statements
  • Drawing commands (circle, rect, moveto, drawto)
  • Custom Methods
  • Array Support
